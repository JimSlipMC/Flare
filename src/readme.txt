Features:

* Potion Limiter & Enchant limiter
* Custom Tab (Potentially)
* Faction System with Claiming Wands, Co Leaders, Economy, Focus, Subclaims, & more.
* Scoreboard with FHome & FStuck timers
* Fully customizable deathban system
* Combat loggers
* Shop Signs
* Combat Tag, PvP Prot, etc
* A custom crates system
* Glowstone Mountain, Koth, Conquest, and a Custom Event (Secrets, when we release I will say what it is.)
* Staff mode with scoreboard integration
* Fully configurable (Language File and Configuration File)
* Classes (Bard, Miner, Archer, Rogue)
* Custom essentials commands (Fly, GMC, GMS, Help, Request / HelpOP, Report, TP, TPHere, TPPos, Bal, Pay, & More)
* Kills, Deaths, and Ore trackers.
* Faction Relation Colors above names.
* Online donator announcement system.
* Fast Smelt
* Mob stacker
* Sign elevators
* Item filter
* Book disenchanter
* Kitmap Option
* Chat management & formatting.
* Worldborder system
* Staff login system
* Blacklist System

Commands:

/flare - Core information command. (Reload functionally - /flare reload)
/l - Lock chat
/delay - Chat delay
/cc - Chat clear
/gmc - Gamemode Creative
/gms - Gamemode Survival
/fly - Fly command
/tp - Teleport command
/tphere - Teleport here command
/tppos - Teleport to a location
/ores - Ores command
/stats - Stats command
/report - Report command
/request - Request command
/focus - Focus command
/togglesb - Toggle scoreboard
/help - Display information about the server
/coords - Display important coordinates

More is coming too :)

Permissions:

You may change the beginning of the permission nodes via the config, instead of flare it will be whatever is in the config.

/flare reload - flare.reload
/l - flare.chat.lock
/delay - flare.chat.delay
/cc - flare.cc
/gmc - flare.gamemode
/gmc <player> - flare.gamemode.other
/gms - flare.gamemode
/gms <player> - flare.gamemode.other
/fly - flare.fly
